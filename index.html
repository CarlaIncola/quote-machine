QUOTE MACHINE

Sometimes it helps with creativity to consider things from another point of view. That's why I've created this random quote machine with +100 phrases from incredible talented artists and great minds.

I designed the interface to give a sober and friendly look, then I developed it to be responsive in all devices. The page is built in a very simple way so that you do not require knowledge of code to be able to intervene. Please feel free to modify it with your favorite authors and colors.
Como parte de mi ruta de aprendizaje en desarrollo front-end, me desafié a mí misma a crear una calculadora funcional de principio a fin.

Diseñé la interfaz de usuario para que funcionara tanto en modo claro como en modo oscuro, y desarrollé toda la aplicación para hacerlo.
